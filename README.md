# Calculadora de Aluguel de Veículos

Script em Python desenvolvido para automatizar o cálculo de faturamento de locação, considerando diárias e quilometragem rodada.

##  Funcionalidades
- **Cálculo de Diárias:** Taxa fixa de R$ 60,00 por dia.
- **Custo por KM:** Taxa variável de R$ 0,15 por quilômetro.
- **Saída Formatada:** Exibição do valor total com duas casas decimais (`.2f`).

##  Conceitos Aplicados
* **Casting:** Conversão de tipos de dados (`int` e `float`).
* **Operadores Aritméticos:** Cálculos de precedência matemática.
* **F-Strings:** Formatação avançada para exibição de valores financeiros.
